	1. Цели и задачи приложения
	Целью разработки приложения является проверка знаний и подготовка учеников средних и старших классов для сдачи ОГЭ по математике 2020-2021 года.
	Задачи приложения:
		Отображение вопросов пользователю.
		Получение ответа пользователя.
		Проверка на правильность ответов.
		Показать правильные и неправильные ответы пользователю в конце 	теста.
		Показать процентное соотношение правильных и неправильных 	вопросов.
		Показать сколько вопросов осталось без ответа.
		Показать среднию оценку по формату ОГЭ по математике 2020-2021 	года.
	2. Портрет целевой аудитории
	Целевой аудиторией приложения являются ученики средних и старших классов, сдающие ОГЭ по математике. С помощью приложения пользователи смогут подготовиться к ОГЭ по математике.
	3. Структура приложения
	Приложение должно быть на русском языке и иметь приятные для глаз, удобные для прочтения тона.
	Приложение должно быть понятным и удобным в использовании.
